Modifier une catégorie de bassin

Vous pouvez mettre à jour le nom et la description de la catégorie sélectionnée.

Points importants:

  • Les modifications affecteront tous les bassins associés à cette catégorie
  • Assurez-vous que le nouveau nom est unique
  • Utilisez une description concise mais informative
